Hi All,

Please see the below update based on our meeting with the architect and landlord this morning:

Building 1
Based on the building specifications and existing tiles, they’ve recommended sticking with framing for the northern and western signage, as there are concerns around the guarantees and potential risks associated with the tiles.

As discussed with Henkes, to ensure the framing is not exposed, we’d like to proceed with the following for these façades:

*	
	Two rails (top and bottom) for the lettering
*	
	A cross frame for the logo
*	
	Please ensure that there are no visible vertical elements included in these frameworks


In addition, for the eastern signage, please update the framing to remove the vertical elements and align it with the previous framework.

Building 3
For Building 3, we require two separate quotes:

1.	
	A quote made out to the landlord to resize the southern signage to fit within the allocated tenant signage zone: 8400mm x 1650mm.
2.	
	There is another façade on Building 3 where signage has been allocated to us. Please provide a quote to PSG for a new sign following the same dimensions as above.
